# EE800_Project_Shrishgovind_Model_and_Data_Driven_Control
This project proposes a hybrid control workflow for a two-wheeled self-balancing robot. First, we use an existing robust model-based control system, which is simulated in MATLAB. Here we can test the model under various dynamic conditions, including collisions and weight additions. Once the model is tested under various input conditions, it will yield a large output sample. These data can be processed to create a data matrix for developing the data-driven control system. A data-driven control system is used in place of a Model-based approach as it is better adaptable to dynamic conditions. With known input-output data, the robot is able to better react to any input in real life. There are 2 types of data-driven control systems: Model-Based and Model-Free. We will conduct experiments on both systems to empirically evaluate which system demonstrates superior performance in power efficiency, computational speed, and space optimization. Passivity-based control employs convex energy shaping to achieve stability by enforcing passivity constraints and ensuring closed-loop energy dissipation during trajectory tracking. To enhance robustness, we will experiment with passivity-based control to achieve stability with minimal energy consumption.
